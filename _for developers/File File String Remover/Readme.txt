This tool allows you to quickly remove text from *all* present files in the same directory. Useful for quickly removing strings such as " 1080p" from downloaded sound files for use as UV Race Music.

---> Requirements <---
1) Python 3.11 or newer

---> How do I use this? <---
1) Place the _removetext.py file in the same directory as the files you want to change.
2) Run the .py file!
3) Type in which strings you'd like to remove, including quotes - " 1080p" for example.
4) Hit Enter and voila!
